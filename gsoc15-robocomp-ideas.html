RoboComp Social Managment

RoboComp is a very complex open source robotics framework, documentation is a very important part of it. We just moved our repositories from sourceforge to github. But we are still in the need of porting some other social tools like the website, the wiki and some tutorials. Clarifiying the usage of RoboComp is an important asset towards achieven a full open source development model that could help us improve the framework and help the open source robotics community. Some work on documenting easing the installation process of the framework is also needed. Achieving an easy to understand documentation and set up would make easier to spread the knowledge about the framework and usage among the open source community.

Current instructions to download, install and start using the software are in Github.
